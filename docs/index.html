Clover Email Manager

A desktop email client built with Godot that allows users to read and send emails through their Gmail account after granting permission through Google OAuth.

Intended to be used in conjunction with a regular email client, this program makes it easier to message multiple people at once.

See conversations between you and someone else in a text chat format and more features.

In its current state, there are features missing, but hopefully you'll get some use from it.

The program needs you to log into your Google account for it to access the email tied to it.
