Information collected

This program accesses all your emails once permission is granted. The user explicitly authorizes the collection of the data through Google OAuth.

The application uses Google Gmail API services to access email data authorized by the user.

How information is used

Email data is used solely to read / display and send emails while using the program. All data usage is within the program while it is running.

Email data is retrieved directly from Google Gmail API services and is not transmitted to any servers controlled by the developer.

Data storage

Email data retrieved by the program is stored in memory while it is running.

For application functionality and organization, some email data, refresh tokens, and other application data may be stored locally on the user's device in JSON format.

No such data is stored on developer-controlled servers.

Data retention

The application stores only the information necessary to provide its functionality, including authentication tokens and any locally saved application data, on the user's device.

The developer does not retain copies of user email data or personal information on developer-controlled servers.

Users may revoke the application's access to their Google Account at any time through their Google Account permissions. Revoking access prevents the application from accessing Gmail but does not automatically remove locally stored data.

Users may delete locally stored application data at any time by deleting the application's data files.

Data sharing

The application does not sell, transfer, or share user email data with third parties.

The developer does not access, monitor, or review user email data.

Data security

The application communicates with Google services using encrypted HTTPS connections provided by Google's APIs.

The application stores its data locally on the user's device and relies on the operating system's security and file permissions to protect that data.

Revoking access

Users may revoke access at any time through their Google Account permissions settings.
